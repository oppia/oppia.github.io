Authenticate with Github

In order to use the Oppia Project Dashboard, you need to provide a personal access token.

  1. Navigate to https://github.com/settings/tokens and login if necessary.
  2. Click "Generate new token"
  3. You can use "Oppia Project Dashboard" as the note.
  4. Select only the "public_repo" checkbox. Then click Generate Token.
  5. Paste the token in the field above and click Login.
